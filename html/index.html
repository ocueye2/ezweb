Welcome to EasyWeb

Easily combine your navigation bar, CSS, HTML, and JavaScript into one streamlined environment.

Getting Started

To fully leverage EasyWeb, a basic understanding of HTML is essential, with optional knowledge of CSS and JavaScript for advanced features.

EasyWeb organizes your project into three essential folders: css, html, and script. These should reside in the same directory as the ezweb.py script.

For instance, if ezweb.py is located in /home, the folders should be in /home/css/, /home/html/, and /home/script/.

Create your HTML, CSS, and JavaScript files and place them in their respective folders. Files with matching names (e.g., pagea.html, pagea.js, pagea.css) will be served at localhost:8080/pagea.

HTML Structure

The index.html file serves as the root of your server at localhost:8080/.

The nav.html file defines your site's navigation bar, automatically appearing at the top of each page.

Global Styles and Scripts

Apply global styling and scripting with all.css and all.js, which affect all pages on your server.

Running the Server

Once your files are ready, launch the server by running ezweb.py. Your website will be accessible at localhost:8080.
